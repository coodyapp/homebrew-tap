# homebrew-tap

Homebrew tap for coodyapp apps.

## Install

```bash
brew tap coodyapp/tap
brew install --cask token-my-bar
```

## Update

```bash
brew update
brew upgrade --cask token-my-bar
```

## Casks

| Cask | Source |
| --- | --- |
| `token-my-bar` | [coodyapp/token-my-bar](https://github.com/coodyapp/token-my-bar) |
